# Requerimientos del Proyecto – Mapa de Capacidades

## Objetivo
Desarrollar una aplicación web que permita visualizar un mapa de capacidades para la empresa Mega Mayorista.

## Requerimientos Funcionales
1. Visualizar las capacidades del negocio.
2. Mostrar las capacidades agrupadas en categorías.
3. Permitir navegar entre niveles de capacidades.
4. Diseñar una interfaz amigable para el usuario.

## Requerimientos No Funcionales
1. Despliegue en la nube.
2. Alta disponibilidad.
3. Bajo costo.
4. Escalabilidad horizontal.
5. Uso de servicios administrados.

## Alcance
Desarrollo del front-end utilizando Next.js, análisis de arquitectura cloud y selección del proveedor.
