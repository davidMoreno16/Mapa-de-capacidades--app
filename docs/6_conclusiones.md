# Conclusiones

La solución propuesta permite desplegar la aplicación del Mapa de Capacidades de manera eficiente, con costos accesibles y cumpliendo los requerimientos de disponibilidad, rendimiento y escalabilidad.

El proceso de comparación entre proveedores cloud permitió identificar que **Google Cloud Platform (GCP)** ofrece mejores condiciones para este proyecto debido a **su menor costo total, su simplicidad de configuración y su buen rendimiento para máquinas de gama estándar**, lo cual lo convierte en la opción más conveniente para la implementación.
